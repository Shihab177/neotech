##Project Introduction
    -The primary objective of this platform is to create a user-friendly online marketplace where users can easily browse products, apply filters, add items to their cart, and place orders.

##Main Features
    #User Side Features
      -Product Browsing
      -Category Based Products
      -Product Details Page
      -Add to Cart System
      -Cart Management (increase/decrease quantity)
      -User Authentication
      -Order Placement

    #Admin Side Features 
      -Product Add / Update / Delete
      -Category Management
      -Order Management

##Technology Stack
    #Frontend
      -HTML
      -Tailwind CSS
      -Next.js
      -Zustand (State Management)

    # UI Enhancements
      -Framer Motion (animations)
      -React Hot Toast (notifications)
      
    #Backend/Database
      -Sanity
